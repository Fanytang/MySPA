# MySPA

> 单页应用演示地址 [demo](https://fanytang.github.io/MySPA).

## 项目运行

``` bash
# 安装依赖
npm install

# 启动服务器在 localhost:8080
npm run dev

# 打包
npm run build

```
## 技术栈

vue2 + vuex + vue-router + axios + mock.js + vue-preview + vue-cli + mint-ui + flex + sass

## 功能实现

* 图片懒加载
* 下拉刷新
* 上拉加载更多
* 图片预览
* 拦截请求
* 底部导航
* 等...
